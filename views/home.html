Image Search Abstraction Layer

Description

A simple API that can retrieve images according to a phrase. These images are given in JSON format.

How to use it

As an example you can search images with the term wizards by typing https://isal-franco.glitch.me/search/wizards?offset=1

You can also paginate the results by changing the offset. For example: https://isal-franco.glitch.me/search/wizards?offset=3

You can also see the history of all the searches done by all the people that have used this API with https://isal-franco.glitch.me/history

Response

The response should be something like: [{"url":"https://www.thesorceryblog.com/powerfulwizard.jpg", "snippet":"Wizards of Wizard Quest", "thumbnail":"https://encrypted-tbn3.gstatic.com/images?q=tbn:ANd9GcQQyv", "context":"https://www.thesorceryblog.com"}, ....... and 9 more relevant images ....... ]
